# Changelog

All notable changes to Double Sync Framework will be documented in this file.

## [1.0.0] - 2024-01-13

### Added
- Initial release of Double Sync Framework
- Core framework with DS global object
- Multi-character system (up to 3 per player)
- Character Selection NUI with western theme
- Modern HUD with status bars and money display
- Economy system (cash, bank, gold)
- Job system with multiple jobs (sheriff, doctor, farmer, miner, etc.)
- Advanced animation system with presets and emotes
- Bidirectional callback system
- Localization support (EN, PT-BR, ES)
- Database module with oxmysql
- Admin commands
- txAdmin recipe for easy deployment

### Core Features
- Player management with state bags
- Auto-save every 5 minutes
- Ban system
- Logging system

### UI/NUI
- Character creation and selection
- HUD with health, hunger, thirst, stamina
- Money display
- Notification system

### Dependencies
- oxmysql (required)
